# Trello API Testing Project

## Project Overview
This project demonstrates API testing of the Trello REST API using Postman.

## Tools Used
- Postman
- Trello API
- Excel

## Objects Tested
- Boards
- Lists
- Cards
- Checklists

## Test Coverage
- Create
- Get
- Update
- Delete
- Verification after deletion

## Key Testing Activities
- Designed request flow with priority and dependency in Excel
- Executed CRUD operations in Postman
- Validated status codes, response time and response body fields
- Used environment variables for request chaining
- Verified deletion of resources after cleanup

## Files Included
- Excel planning sheet
- Postman collection export
- Postman environment export
- Postman test run result export
- Screenshots of execution
